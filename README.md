# Iнструкцiя з запуску

## MongoDB

* Запустити mongod 

## Програма

* Створіть віртуальне оточення Python 
>> python virtualenv venv

* Активуйте віртуальне оточення Python
>> Scripts/activate.bat

* Встановіть необхідні залежності
>> pip install -r requirements.txt

* Запустіть програму
>> python main.py
